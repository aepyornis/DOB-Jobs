This table and database contains all 'jobs' from the Department of Buildings in 2014. A job or application can contain multiple permits. The source for the data comes from DOB monthly reports, available here. Note that this contains jobs and applications that were denied, changed, and/or never built.

This site is a work in progress. Please check back regularly for more updates and features, including the ability to download reports, visualizations, and permit data.

The fields shown above are only a selection from the 60+ fields in the original spreadsheet.
